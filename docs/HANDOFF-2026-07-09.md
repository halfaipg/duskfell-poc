## Run + screenshot recipe

```bash
cargo run -p sundermere-server          # http://127.0.0.1:4107
"/Applications/Google Chrome.app/Contents/MacOS/Google Chrome" \
  --headless=new --disable-gpu --hide-scrollbars \
  --user-data-dir=/tmp/fresh-profile \
  --window-size=1280,800 --virtual-time-budget=15000 \
  --screenshot=shot.png "http://127.0.0.1:4107/"
```
Fresh profile per shot (stale asset cache otherwise). Server re-verifies
manifests at startup — restart after asset/manifest changes.
`?terrainDebug=material|zone|elevation` colors tiles by property.

## Resolved: pale-cream "X" ribbons

The replacement elevation-aware patch mapping removed the long pale X
ribbons. Patched tiles now use exact per-triangle terrain projection and skip
the old atlas/facet/height/relief overlays that scratched visible diamond lines
across the painting. Side walls remain for real elevation drops.
